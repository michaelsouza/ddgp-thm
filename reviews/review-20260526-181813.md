## Major Issues

### Issue 1: Generic Mirror Separation Reads Like an Unproved Theorem

* **Severity**: Major
* **File & Line**: `article/sections/shifts.tex` (Line 16), `article/sections/scope.tex` (Lines 8-10), `article/sections/results.tex` (Line 72)
* **Tsitsiklis Basis**: Broad mathematical style: define assumptions before use; avoid hiding hard work inside definitions; make theorem statements crisp and honest about hypotheses.
* **Problem**: Definition~\ref{def:mirror-sep} contains the exact converse needed by Theorem~\ref{thm:main}. This is acceptable as a hypothesis, but the current wording can make a reader expect a proof that the exceptional set is proper for the constructed violation matrix. The phrase "algebraic bracket-separation arguments used to establish the generic presentation lemma" also refers to a lemma or argument that is not present in the manuscript.
* **Original LaTeX**:

```latex
For $K=1$, the algebraic bracket-separation arguments used to establish the generic presentation lemma for $K \ge 2$ degenerate on the line, so that case requires separate treatment.
```

* **Recommended Replacement**:

```latex
For $K=1$, mirror separation has a different geometry on the line, and we do not claim that the present hypothesis captures all one-dimensional exceptional cases. The theorem is therefore stated only for $K \ge 2$.
```

* **Rationale**: This removes a reference to a missing proof object. If the paper is meant to prove generic mirror separation rather than assume it, add a separate lemma before Theorem~\ref{thm:main}; otherwise, add one sentence after Definition~\ref{def:mirror-sep} saying that the theorem uses this as an explicit genericity hypothesis.

### Issue 2: Combinatorial DDGP Recovery Needs a Global-Active-Set Bridge

* **Severity**: Major
* **File & Line**: `article/sections/scope.tex` (Lines 15-36), with dependency on `article/sections/subproblems.tex` (Lines 30-41)
* **Tsitsiklis Basis**: Minimize reader effort; remind the reader of distant definitions; avoid sudden changes in scope or notation.
* **Problem**: The local construction defines `L_P` from a pruning set `P`. In the no-pruning Combinatorial DDGP case, taking `P=\varnothing` would give an empty pruning-induced local set, not the global count `2^{n-K}`. The special-case discussion is mathematically plausible, but it needs an explicit bridge saying that the same generator/rank construction is being applied to the global active set `L=V`, `B=V\setminus V_0`, and `F=E_D`.
* **Original LaTeX**:

```latex
The rank-count formula includes this result as a direct special case:
\begin{enumerate}
```

* **Recommended Replacement**:

```latex
To compare with the no-pruning count, apply the same generator and rank construction to the global active set
\[
L=V,\qquad B=V\setminus V_0,\qquad F=E_D.
\]
With this global active set, the rank-count formula includes the Combinatorial DDGP result as a direct special case:
\begin{enumerate}
```

* **Rationale**: This keeps the special case from looking inconsistent with the earlier pruning-local definition. It also reduces the chance that a reviewer asks whether `P=\varnothing` makes `L_P` empty.

* **Severity**: Minor
* **File & Line**: `article/sections/subproblems.tex` (Line 6)
* **Tsitsiklis Basis**: Break long mathematical sentences; define symbols close to first use.
* **Problem**: The paragraph introduces the vertex order, subscript convention, seed simplex, seed predecessor sets, non-seed predecessor sets, and dependency DAG in one long block.
* **Original LaTeX**:

```latex
Let the vertices be ordered as $v_1,\ldots,v_n$, and write $U_i$ for the predecessor set of $v_i$. When a vertex $q=v_i$ appears as a subscript, $U_q$ means $U_i$; this preserves the paper's vertex notation while making the order explicit. Let $V_0=\{v_1,\ldots,v_{K+1}\}$ be the fixed initial simplex in $\mathbb{R}^K$. For $i\le K+1$, set $U_i=\varnothing$. For every non-seed vertex $i > K+1$, the predecessor set $U_i \subseteq \{v_1,\ldots,v_{i-1}\}$ has size $K$. The predecessor relation defines a directed acyclic graph with an arc $v_j \to v_i$ whenever $v_j \in U_i$.
```

* **Recommended Replacement**:

```latex
Let the vertices be ordered as $v_1,\ldots,v_n$, and write $U_i$ for the predecessor set of $v_i$. When a vertex $q=v_i$ appears as a subscript, $U_q$ means $U_i$; this preserves the paper's vertex notation while making the order explicit.

The predecessor relation defines a directed acyclic graph with an arc $v_j \to v_i$ whenever $v_j \in U_i$.

* **Rationale**: This is only a paragraph break, but it makes the model easier to parse.
