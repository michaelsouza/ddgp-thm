# DDGP Rank-Count Slides

Modern Reveal.js deck for `research/ddgp-rank-count-theory-report.md`.

Open `slides/index.html` in a browser. The deck uses Reveal.js and MathJax from
CDN links, so an internet connection is needed for the presentation framework
and math rendering.

Each slide keeps the argument short and includes links to the glossary concepts
used on that slide.
