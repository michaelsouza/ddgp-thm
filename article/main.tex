\begin{abstract}
In the Discretizable Distance Geometry Problem (DDGP), nonconsecutive
predecessor sets prevent a global solution count that depends only on graph
topology. We consider the local counting problem obtained by fixing an active
set of pruning constraints in a generic DDGP subproblem. We show that,
relative to any reference solution, the feasible binary Branch-and-Prune codes
form an affine space over \(\mathbb{F}_2\). Its direction is determined by the
branch shifts that produce no labeled pruning-edge violation. Consequently, the
local solution count follows from a rank computation, without enumerating
Branch-and-Prune branches. The formula recovers the known clique-based counts
for the Discretizable Molecular Distance Geometry Problem and for
Combinatorial DDGP instances.
\end{abstract}
