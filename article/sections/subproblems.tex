\section{Local DDGP Subproblems}
\label{sec:subproblems}

Let
\[
V_0=\{v_1,\ldots,v_{K+1}\}
\]
be the fixed initial simplex in $\mathbb{R}^K$. For every non-seed vertex $i > K+1$, the predecessor set $U_i$ has size $K$. The predecessor relation defines a directed acyclic graph with an arc $u \to i$ whenever $u \in U_i$. Related order-dependent reference structures have been studied through $K$-discretization and $K$-incident graphs \cite{abud2018k}; here we fix one local predecessor system and derive branch shifts from its dependency cones.

\begin{definition}[Fixing set]
\label{def:fixing}
The fixing set of a vertex $i$ is
\[
\operatorname{Fix}_U(i)
=
\{i\}\cup \bigcup_{u\in U_i}\operatorname{Fix}_U(u).
\]
\end{definition}

\begin{definition}[Dependency cone]
\label{def:cone}
The dependency cone of a vertex $i$ is
\[
\operatorname{Cone}_U(i)
=
\{i\}\cup \bigcup_{w:\,i\in U_w}\operatorname{Cone}_U(w).
\]
\end{definition}

For a set of pruning edges $P \subset E_P$, define the local vertex set
\[
L_P
=
\bigcup_{\{u,v\}\in P}
\left(\operatorname{Fix}_U(u)\cup \operatorname{Fix}_U(v)\right),
\]
and the local binary decision set
\[
B_P=L_P\setminus V_0.
\]
The set $B_P$ contains the branch decisions that can affect the positions of the endpoints of the pruning constraints in $P$. Restricting our analysis to $L_P$ keeps the construction local: vertices outside $L_P$ do not enter the branch code being counted.

The active edge set is denoted by $F$. In the main theorem, $F$ must contain the local discretization/base edges $E_D[L_P]$. In applications, one usually takes
\[
F = E_D[L_P] \cup P.
\]
The local solution code is
\[
\Xi_F
=
\{s \in \mathbb{F}_2^{B_P} \mid s \text{ realizes every edge in } F\}.
\]
Our goal is to compute the exact cardinality $|\Xi_F|$ under generic geometric assumptions without running tree-based branch search.
