\section{Conclusion}
\label{sec:conclusion}

We developed a rank-count theory for generic local solution sets in the Discretizable Distance Geometry Problem. The central point is that the obstruction in general DDGP counting is not the absence of branch structure; it is the loss of unlabeled global cancellation. Once violations are labeled by their mirror cliques, feasible local branch differences become the image of a kernel over $\mathbb F_2$.

The Feasible Branch Shift Theorem identifies the local solution code from any reference solution as the affine coset
\[
\Xi_F = s^\ast + M(\ker V),
\]
and the rank-count lemma computes its size using only ranks over $\mathbb{F}_2$. The method is local and generic: it counts fixed active DDGP subproblems with non-consecutive predecessor structure outside a proper algebraic exceptional set.

Several questions remain open. One is how to select active subproblems automatically inside a larger Branch-and-Prune computation. Another is how overlapping local counts can be combined without losing the mirror labels that make cancellation valid. A third is how to characterize, detect, or exploit the exceptional nongeneric cases where additional coincidences create more solutions than the generic rank count predicts.
