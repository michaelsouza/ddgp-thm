\section{Feasible Branch Shifts}
\label{sec:shifts}

This section defines the feasible branch shift space. These are the branch transformations produced by generator combinations whose net labeled violation is zero.

Generator combinations with zero net labeled violation represent branch transformations that preserve all active edges. The feasible branch shift space is
\[
\mathcal{K}_F = M(\ker V).
\]
Thus, $\mathcal{K}_F$ consists of branch shifts that can be produced by zero-violation generator combinations.

The distinction between presentations and branch masks is essential here. Elements of $\ker V$ are generator presentations, not branch masks. Different generator combinations may produce the same branch mask. If $M\alpha = M\beta$, then $\alpha + \beta \in \ker M$. The object to count is therefore the image $M(\ker V)$, not the set of presentations $\ker V$.

\begin{definition}[Generic mirror separation]
\label{def:mirror-sep}
View the local predecessor data and the edge lengths in $F$ as parameters on the real algebraic set where all required lateration steps are nondegenerate. The local active subproblem satisfies generic mirror separation if there is a proper algebraic exceptional set $Z$ such that, for every parameter choice outside $Z$, every shift $h\in\mathbb{F}_2^{B_P}\setminus\mathcal{K}_F$, and every local branch code $s\in\mathbb{F}_2^{B_P}$ for which the lateration embeddings $x(s)$ and $x(s\oplus h)$ are defined, the two embeddings assign different lengths to at least one active edge in $F$.
\end{definition}

The next section uses this definition to prove the algebraic rank count.
