\section{Feasible Branch Shifts}
\label{sec:shifts}

This section formalizes the concept of the feasible branch shift space, which characterizes the coordinate transformations that preserve active distance constraints. 
Geometrically, moving from a branch code $s$ to $s \oplus h$ via a branch shift $h \in \mathbb{F}_2^{B_{\mathrm{c}}}$ corresponds to a sequence of partial reflections. 
A shift preserves the validity of all active constraints if it is produced by a generator combination with zero net violations, meaning $\alpha \in \ker V$. 
Because multiple generator combinations can yield the same branch mask, the space of valid branch shifts $\mathcal{K}_F$ is defined as the image of this kernel under the mask matrix $M$:
\[
\mathcal{K}_F = M(\ker V).
\]
To ensure that only shifts within $\mathcal{K}_F$ preserve these distances under generic geometric configurations, we introduce the \emph{generic mirror separation} hypothesis.

\begin{definition}[Generic mirror separation]
\label{def:mirror-sep}
Fix the ordered DDGP template, including the predecessor sets and the active edge set $F$. Thus $M$, $V$, and $\mathcal{K}_F=M(\ker V)$ are fixed. Let $\Theta$ be the nondegenerate parameter domain, viewed as a Zariski-open semialgebraic subset of the corresponding real parameter variety.

For $\theta\in\Theta$, write $x^\theta(s)$ for the embedding associated with a branch code $s$. We say that $\theta$ is mirror-separated relative to $F$ if, for every shift $h \notin \mathcal{K}_F$ and every branch code $s$ for which both embeddings $x^\theta(s)$ and $x^\theta(s\oplus h)$ are defined, there exists an active edge $\{a,b\} \in F$ such that
\[
\|x^\theta_a(s \oplus h) - x^\theta_b(s \oplus h)\|_2^2 \neq \|x^\theta_a(s) - x^\theta_b(s)\|_2^2.
\]
The template satisfies generic mirror separation if the parameters that are not mirror-separated are contained in a relative algebraic subset $Z \subset \Theta$ that is proper on each irreducible component under consideration.
\end{definition}

Generic mirror separation is a genericity hypothesis on the numerical parameters of a fixed ordered template. It rules out accidental algebraic coincidences in which a shift $h\notin\mathcal{K}_F$ nevertheless preserves all active edge distances. When the exceptional set $Z$ is proper on the relevant component of $\Theta$, it has relative measure zero there; in this precise sense the hypothesis holds for almost every nondegenerate continuous choice of parameters. The main theorem below is stated for mirror-separated parameter choices and does not assert that every ordered template satisfies this genericity condition.

The next section uses this hypothesis to prove the algebraic rank count.
