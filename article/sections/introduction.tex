\section{Introduction}
\label{sec:intro}

This paper studies a local counting question in the Discretizable Distance Geometry Problem (DDGP): after fixing a set of active pruning constraints, how many Branch-and-Prune branch codes remain feasible? For DMDGP instances, consecutive predecessor sets give partial-reflection symmetries and combinatorial power-of-two counts. For general DDGP instances, predecessors need not be consecutive, so a branch change propagates through dependency cones and may preserve or violate active distances depending on the mirror clique. We show that, under a generic separation assumption, the local count is still computable by linear algebra over $\mathbb F_2$.

The Distance Geometry Problem (DGP) asks for a geometric representation of a weighted graph in $\mathbb{R}^K$: given a simple undirected graph $G = (V, E)$ and an edge weight function $d: E \to \mathbb{R}_+$, find an embedding $x: V \to \mathbb{R}^K$ such that $\|x_i - x_j\|_2 = d_{ij}$ for every $\{i,j\}\in E$ \cite{mucherino2012discretizable}. The problem appears in protein structure determination from nuclear magnetic resonance (NMR) distance data \cite{liberti2008branch}, sensor network localization \cite{abud2018k}, and the geometry of nanostructures \cite{mucherino2020analysis}.

When a vertex order on $V = \{1, \ldots, n\}$ is part of the input, the search space can be discretized. In DDGP and DMDGP instances \cite{mucherino2012discretizable,lavor2012discretizable}, each non-seed vertex $j > K+1$ has a predecessor set $U_j \subset \{1, \ldots, j-1\}$ of size $K$. Once the predecessors are placed, the position of $j$ is obtained by $K$-lateration, the intersection of $K$ spheres centered at the predecessor coordinates \cite{liberti2014number}. Generically, this intersection contains at most two points, so realizations can be organized as paths in the Branch-and-Prune (BP) tree \cite{liberti2008branch}.

The edges in $G$ are naturally partitioned into two sets:
\begin{enumerate}
    \item \textbf{Discretization edges} ($E_D = \{\{i, j\} \in E \mid i \in U_j\}$), which are used to perform the $K$-lateration steps and branch into the binary tree.
    \item \textbf{Pruning edges} ($E_P = E \setminus E_D$), which represent additional distance constraints that are used by the BP algorithm to check the feasibility of branched positions and prune inconsistent paths in the tree.
\end{enumerate}

General DDGP counting cannot be purely graph-combinatorial in the global sense: numerical edge weights can change the number of feasible branches \cite{abud2024impossible}. We therefore work locally. Around a fixed pruning set, we build the active subproblem, encode branch transformations by graph-derived generators, and record which active edges each generator can violate. The generic mirror-separation assumption excludes the proper algebraic exceptional set where a geometrically violating transformation accidentally preserves a distance. Outside this set, the feasible branch differences are exactly the zero-violation shifts.

Our contribution is threefold:
\begin{enumerate}
    \item We define local active subproblems around a pruning set and represent their branch transformations using graph-derived cone generators and base generators.
    \item We introduce a labeled violation matrix that separates violations caused by different mirror cliques, keeping track of mirror-specific incompatibilities.
    \item We prove the \emph{Feasible Branch Shift Theorem}: if $s^\ast$ is a reference solution, then
    \[
    \Xi_F=s^\ast+\mathcal K_F,
    \qquad
    \mathcal K_F=M(\ker V).
    \]
    Rank-nullity then gives the count from
    \[
    \dim \mathcal{K}_F = \operatorname{rank}\begin{bmatrix}M\\ V\end{bmatrix} - \operatorname{rank}(V)
    \]
    where $M$ is the generator mask matrix and $V$ is the labeled violation matrix.
\end{enumerate}

All vector spaces, sums, kernels, and ranks discussed below are defined over the finite field $\mathbb{F}_2$. Binary branch-code representations are standard for paths in the Branch-and-Prune tree; related binary representations have been widely used to analyze symmetry and degrees of freedom in distance geometry \cite{mucherino2012exploiting,mucherino2020analysis}.

The remainder of the paper is organized as follows. Section~\ref{sec:related} reviews DMDGP symmetries, DDGP recognition, and the counting barrier for general DDGP instances. Sections~\ref{sec:subproblems}--\ref{sec:shifts} define the local subproblem, generators, labeled violations, and feasible branch shifts. Section~\ref{sec:results} proves the main theorem and rank-count lemma. Section~\ref{sec:example} gives a fully worked 6-vertex example. Section~\ref{sec:scope} discusses scope, genericity, and the Combinatorial DDGP special case.
