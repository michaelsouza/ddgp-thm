## Intuition

The discretization field is an algebraic way to describe the binary branching
of the Branch-and-Prune construction.

For a generic DDGP instance, each non-seed vertex is obtained by intersecting
$K$ spheres. Once its predecessor clique is fixed, this creates one quadratic
choice: the point lies on one side or the other side of the predecessor
hyperplane. Algebraically, this is a square-root sign.

Thus the DDGP search tree can be viewed as a finite iterated quadratic cover.
The full splitting field of such a tower need not have an abelian Galois group,
but the Branch-and-Prune tree still has a natural binary branch-code labelling.
This viewpoint is useful for proving solution-count formulas because active
edge lengths become algebraic functions on the cover, and valid branch
differences are exactly the sign changes that preserve those functions.

## Formal definition

Fix a DDGP order, predecessor sets $U_i$, and an initial rigid simplex

$$
V_0=\{v_1,\dots,v_{K+1}\}.
$$

Let $\mathbb K_0$ be the field generated by the generic coordinates of the
initial simplex and by the symbolic distances used by the discretization
construction.

For each non-seed vertex $v_i$, the $K$-lateration step writes the new point as

$$
x_i=p_i+h_i n_i,
$$

where:

- $p_i$ lies in the affine span of the predecessor clique $U_i$;
- $n_i$ is a unit normal to $\operatorname{aff}(U_i)$;
- $h_i^2$ is determined by the sphere equations.

Generically, $h_i$ is a quadratic sign over the field generated by the previous
vertices. The discretization field is the tower

$$
\mathbb K_0
\subset
\mathbb K_{K+2}
\subset
\cdots
\subset
\mathbb K_n,
$$

where $\mathbb K_i$ is obtained by adjoining the coordinates of $v_i$.

Outside discriminant-zero exceptional cases, each non-seed vertex contributes a
degree-two choice. The full splitting field of the iterated tower can have
nonabelian monodromy, so one should not identify it automatically with an
elementary abelian Galois group. What the DDGP order gives canonically is the
branch-code torsor

$$
\mathbb F_2^{V\setminus V_0}.
$$

For a local DDGP subproblem with decision set $B$, this restricts to
$\mathbb F_2^B$.

## Use in solution counting

For an active edge $e=\{u,v\}$, the squared edge length

$$
\lambda_e=\|x_u-x_v\|^2
$$

is an element of the discretization field.

A branch difference preserves the edge if and only if the corresponding pair of
points on the BP cover has the same value of $\lambda_e$. Therefore, for an
active edge set $F$ and a fixed branch code $s$, the local edge-preserving
differences are

$$
\{h\in\mathbb F_2^B
\mid
\lambda_e(s\oplus h)=\lambda_e(s)
\text{ for every } e\in F\}.
$$

The [labeled presentation property](labeled-presentation-property.md) can be
read as a statement about the character support of the elements $\lambda_e$:
the active edge-length functions should factor through the elementary-abelian
character quotient recorded by the [labeled violation kernel](labeled-violation-kernel.md).
