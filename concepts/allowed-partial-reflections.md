## Intuition

A hallmark result of the [DMDGP](discretizable-molecular-distance-geometry-problem.md) is that the number of incongruent valid realizations of a YES instance is always a power of two with probability 1. This property is mathematically established by analyzing the algebraic group formed by **Allowed Partial Reflections**.

When solving the DMDGP using the [Branch-and-Prune (BP) Algorithm](branch-and-prune.md), the search space is represented as a binary tree where each node represents a candidate position of a vertex. A branching occurs at level $i$ (vertex $v_i$) because there are two candidate coordinates that satisfy the discretization distances to its $K$ predecessors. 

A **partial reflection** $g_i$ corresponds to reflecting the coordinates of all subsequent vertices $v_i, \dots, v_n$ across the local hyperplane defined by the predecessors of $v_i$. If both branches of the search tree starting from a node at level $i$ can be extended all the way to feasible leaf nodes (representing full valid realizations), we call $g_i$ an **allowed partial reflection**. 

The set of all allowed partial reflections generates an abelian subgroup of the binary group $C_2^n$ under modulo-2 addition. Since any valid realization can be mapped to any other valid realization through a unique combination of allowed partial reflections, the total number of solutions is exactly equal to the order of this symmetry group, which is mathematically guaranteed to be a power of two.

## Formal definition

Let $\mathcal{T} = (\mathcal{V}, \mathcal{A})$ be the binary search tree generated by the BP algorithm for a DMDGP instance, where nodes $\alpha \in \mathcal{V}$ represent partial embeddings $x(\alpha)$. Let $\mathcal{V}_i$ denote the set of nodes at level $i$, and let $\mu(\alpha) = \boxplus$ indicate that a node is feasible.

Let $I \subseteq \{1, \dots, n\}$ be the set of levels in $\mathcal{T}$ for which, for every feasible node $\alpha \in \mathcal{V}_{i-1}$, both of its children are feasible and both have paths leading to feasible leaf nodes in $\mathcal{V}_n$. 

Let $\mathcal{G} = (\mathbb{F}_2^n, \oplus) \cong C_2^n$ be the additive group of binary sequences under elementwise modulo-2 addition. For each level $i \in N = \{1, \dots, n\}$, let $g_i \in \mathbb{F}_2^n$ be the binary row vector representing a partial reflection at level $i$, where the $j$-th component is:

$$
g_{ij} = \begin{cases} 
1 & \text{if } i \le j \\ 
0 & \text{otherwise} 
\end{cases}
$$

Let $\Gamma = \{g_i \mid i \in N\}$ be the set of all partial reflections. The **group of allowed partial reflections** $\Lambda$ is the subgroup of $\mathcal{G}$ generated by the allowed levels $I$:

$$
\Lambda = \langle \{ g_i \in \Gamma \mid i \in I \} \rangle
$$

By algebraic properties of binary generator matrices:
1. The mapping $\phi: \mathcal{P}(N) \to \mathcal{G}$ defined by $\phi(S) = \bigoplus_{i \in S} g_i$ is injective.
2. The cardinality of the group $\Lambda$ is:
   $$|\Lambda| = 2^{|I|}$$
3. **Orbit-Stabilizer / Symmetry Theorem**: If the DMDGP instance is a YES instance, then the set of all valid embeddings in their binary path representation, $\Xi \subseteq \mathbb{F}_2^n$, is a single orbit under the action of $\Lambda$. That is, for any valid path $\xi \in \Xi$:
   $$\xi \oplus \Lambda = \Xi \quad \text{with probability 1}$$

Consequently, the number of solutions is:
$$|X| = |\Xi| = 2^{|I|} \quad \text{with probability 1}$$
